# Wethlete Brand Styling

## Overview
To access Wethlete's official brand identity and style resources, use this skill.

Keywords: branding, corporate identity, visual identity, compression, performance, recovery, circulation, styling, brand colors, typography, visual formatting, visual design

## Brand Guidelines

### Colors

Main Colors:

Black: #030303 - Primary text and dark backgrounds  
White: #F2F2F2 - Light backgrounds and text on dark backgrounds

Accent Colors:

Orange: #F1831D - Primary accent representing energy, circulation, and activation

### Typography

Headings: Instrument Serif (with Georgia fallback)  
Body Text: PP Neue Montreal Book / Medium (with Arial fallback)

Note: Fonts should be pre-installed in your environment for best results.

## Features

### Smart Font Application

Applies Instrument Serif font to headings (24pt and larger)  
Applies PP Neue Montreal font to body text  
Automatically falls back to Georgia for headings and Arial for body text if custom fonts are unavailable  
Preserves readability across all systems

### Text Styling

Headings (24pt+): Instrument Serif font  
Body text: PP Neue Montreal font  
Short, direct sentences  
Clear hierarchy and spacing  
Smart color selection based on background  
Preserves readability, contrast, and structured visual hierarchy

### Shape and Accent Colors

Non-text shapes use the primary accent color  
Orange (#F1831D) should be used for highlights, CTAs, and key focal elements  
Maintains visual focus without overwhelming the layout

## Technical Details

### Font Management

Uses system-installed Instrument Serif and PP Neue Montreal fonts when available  
Provides automatic fallback to Georgia for headings and Arial for body text  
No font installation is required if fallback fonts are acceptable  
For best results, pre-install Instrument Serif and PP Neue Montreal in your environment

### Color Application

Uses HEX color values for precise brand matching  
Black: #030303  
White: #F2F2F2  
Orange: #F1831D

Apply dark backgrounds with high-contrast light text when possible  
Use orange sparingly for emphasis and conversion-focused elements  
Maintains color fidelity across different systems

## Design Principles

Design communicates function first

Compression should be presented as support, circulation, recovery, and performance , a tool that makes life better

Avoid decorative, soft, or unnecessary visual elements

Every element must reinforce clarity, purpose, and strength

## Messaging Alignment

Focus on functional outcomes

Reduce fatigue  
Improve circulation  
Support muscle function  
Enhance recovery

Avoid vague, lifestyle-only, or generic wellness claims

Be specific, direct, and benefit-driven

## System Behavior

Design should feel controlled, strong, and precise

Visual output should avoid softness, clutter, and generic fitness styling

The brand should communicate confidence, function, and modern performance

## Final Rule

If the design does not communicate improved performance and function instantly, it is wrong.
